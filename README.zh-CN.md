# 微信小程序文档扫描工具

轻量级、离线运行的长方形文档裁剪工具。提供默认5%内缩框 + 拖角/拖边/平移手动调整。自动边界检测为实验性功能；手动调整是稳定的主流程。

**不是**：OCR 工具。不进行透视矫正。不调用服务器。

---

## 功能状态 (v0.1.0)

| 功能 | 状态 |
|------|------|
| 自动边界检测 | **实验性** — 高对比度图像效果较好 |
| 手动拖角 | **稳定** |
| 拖边 | **稳定** |
| 整体平移 | **稳定** |
| 单页裁剪 | **稳定** |
| 多页管理 | **演示** |
| 透视矫正 | **未实现** |
| OCR | **不包含** |

> 自动边界检测为实验性功能。在12个合成场景测试中，仅高对比度（白纸+深色背景）产生自动检测结果，其余回退到5%默认框。回退属于**正常设计**，不是失败。手动调整是可靠的主工作流。

---

## v0.1.0 功能

- **5%默认内缩框** — 无需检测即可使用
- **720px缩略图分析** — 缩小版快速分析
- **手动调整** — 拖角、拖边、整体平移
- **超时回退** — 检测超时自动退回默认框
- **用户优先** — 手动移动后不再被自动检测覆盖
- **单页/多页** — 支持一页或多页
- **Canvas导出** — 最大3000px（仅矩形）

## 快速接入

将 `miniprogram-component/document-cropper/` 复制到小程序项目中，注册组件即可使用（详见 [英文README](README.md)）。

## 组件 API

| Properties | 类型 | 说明 |
|-----------|------|------|
| `src` | String | 图片临时文件路径 |
| `imageInfo` | Object | `{width, height}` |
| `cropPoints` | Array | 覆盖初始裁剪点 |

| Events | detail | 说明 |
|--------|--------|------|
| `ready` | `{}` | 组件就绪 |
| `detectstart` | `{}` | 检测开始 |
| `detectcomplete` | `{points,confidence}` | 检测完成 |
| `detectfallback` | `{reason}` | 回退到默认框 |
| `change` | `{points}` | 拖动中 |
| `crop` | `{points}` | 拖动结束 |
| `error` | `{message}` | 错误(输入无效、Canvas失败等) |

| Methods | 说明 |
|---------|------|
| `getPoints()` | 获取当前裁剪点 |
| `setPoints(points)` | 设置裁剪点 |
| `exportCrop()` | 导出裁剪图 Promise |

## 演示项目

目录 `examples/wechat-miniprogram-demo/` 包含最小示例，支持选图→裁剪→导出→多页。不接服务器、不接OCR。

导入方式：微信开发者工具 → 新建项目 → 指向 `examples/wechat-miniprogram-demo/` → 替换 `appid`。

## 已知限制

- 仅矩形裁剪，不含透视矫正
- 不使用 `onCameraFrame` 实时检测
- 自动检测为实验性，复杂背景需手动调整
- 非专业扫描 SDK

## 隐私

全部在设备本地处理，不发送图像数据到任何服务器。
